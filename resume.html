EECS 482

Introduction to Operating Systems

more info...

Study of operating systems and low-level functionality. EECS 482 covers multithreading and concurrency, virtual memory, file systems, virtual machines, and remote procedure calls. Projects in EECS 482 include:

  • Disk Scheduler: implementation of a multithreaded producer-consumer queue for servicing disk requests, using a shortest seek time first (SSTF) algorithm.
  • Thread Library: implementation of various multithreading utilities, such as threads, mutexes, and condition variables, as well as handling context switches between threads and blocking threads.
  • Virtual Memory: implementation of a memory management unit (MMU) to manage virtual memory inside a portion of a process's address space. Tests understanding of the implementation of virtual memory, including page tables, paging in and out from disk, and the clock eviction algorithm.
  • File Server: implementation of a file server working with inodes on disk to represent files. Tests understanding of the underlying implementation of file systems, as well as the usage of Berkeley sockets.

Introduction to Computer Security

more info...
